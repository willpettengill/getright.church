# getright.church

A church-related web application built with Next.js and Supabase.

## Setup

1. Clone the repo
2. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the dev server:
   ```bash
   npm run dev
   ```

## Tech Stack

- **Next.js** — React framework (via Vercel v0)
- **Supabase** — Auth, Database, Storage, Edge Functions
- **TypeScript**
